# Hot Paper WeChat Publish

本技能用于生成高质量的科技行业热点日报和AI行业热点日报，并自动发布到微信公众号草稿箱。

## 功能概述

- **热点检索**：全面检索科技领域和AI领域的当前热点事件
- **信息采集**：深入搜集每个热点话题的详细信息
- **系统整理**：系统化整理与专业化分析信息
- **日报生成**：生成结构清晰、内容详实的热点日报
- **自动发布**：自动发布到微信公众号草稿箱

## 环境准备

### 1.1 安装 wenyan-cli

wenyan-cli 是微信公众号发布的命令行工具。

```bash
# 使用 npm 全局安装
npm install -g wenyan-cli

# 验证安装
wenyan --version
```

### 1.2 配置微信公众号

```bash
# 设置 AppID 和 AppSecret
export WECHAT_APP_ID="你的AppID"
export WECHAT_APP_SECRET="你的AppSecret"
```

> **提示**：AppID 和 AppSecret 需要在微信公众平台后台获取。登录微信公众平台 → 设置与开发 → 基本配置。

### 1.3 创建必要的目录

```bash
# 创建日报输出目录
mkdir -p {skill_dir}/daily_reports
```

## 执行流程

### 第一阶段：热点检索与采集

#### 1.1 检索热点事件

使用 `trends-hub` 工具全面检索当前热点事件：

**筛选标准**：

- 科技领域相关热点
- AI领域相关热点
- 热度值较高的话题
- 影响力较大的事件

#### 1.2 监控官方渠道

读取并监控以下官方渠道的最新消息：

**信息源文件**：`{skill_dir}/assets/source_links.md`

**监控范围**：

**国际科技巨头**：Apple、Microsoft、Google、Amazon、Meta、NVIDIA、Samsung、IBM、Intel、TSMC

**国内科技巨头**：百度、华为、腾讯、阿里巴巴、京东、字节跳动

**国际AI公司**：OpenAI、Google DeepMind、Anthropic、Mistral AI、Cohere

**国内AI公司**：百度文心、智谱 AI、月之暗面 AI、阶跃星辰 AI、零一 AI

**科技媒体**：36Kr、TechNode、GeekPark、雷锋网、新浪科技、TechCrunch、The Verge、Wired、爱范儿、InfoQ

**监控方法**：

1. 从信息源文件（`{skill_dir}/assets/source_links.md`）读取官方链接
2. 使用 运用 `WebSearch` 或`WebFetch` 工具获取各官网的最新动态
3. 筛选出 24 小时内发布的重要消息

#### 1.3 深度信息采集

针对筛选出的每个热点话题，运用 `WebSearch` 和 `WebFetch` 工具深入搜集信息：

**工具使用策略**：

| 工具          | 用途             | 示例            |
| ----------- | -------------- | ------------- |
| `WebSearch` | 广泛搜索相关新闻、报道、评论 | 搜索全网对事件的报道    |
| `WebFetch`  | 抓取具体网页的详细内容    | 获取官网公告、深度文章全文 |

**搜索维度**：

- **事件背景**：事件起因、发展历程
- **核心技术**：技术原理、创新点
- **行业影响**：对行业的影响、市场反应
- **专家观点**：行业专家评价、分析师观点
- **最新动态**：最新进展、后续计划

**操作流程**：

1. 使用 `Tavily Search` 广泛搜索
2. 使用 `Tavily Extract` 抓取关键信息源
3. 信息交叉验证

### 第二阶段：信息处理与分析

#### 2.1 信息去重与验证

- 去除重复信息
- 交叉验证信息来源（至少2个独立来源）
- 确保信息准确性

#### 2.2 分类整理

**按领域分类**：

- 科技综合
- AI技术
- 产品发布
- 投融资
- 行业政策

**按重要性分级**：

- S 级（重大）：行业颠覆性事件、巨头核心发布
- A 级（重要）：重要产品更新、重大合作
- B 级（一般）：常规动态、小额融资

#### 2.3 专业分析

- 事件背后的技术趋势
- 对行业格局的影响
- 未来发展方向预测

### 第三阶段：日报编撰与发布

#### 3.1 内容编撰

选择热度最高、影响力最大、最有价值的**10**条事件，分别编撰成：

**科技热点日报** 和 **AI 热点日报**

#### 3.2 日报格式模板

**Front Matter 元信息**（文章开头必须包含）：

```markdown
---
title: 科技热点日报|YYYY-MM-DD
cover: {skill_dir}/asset/科技日报.png
---
```

> **提示**：AI热点日报请使用 `cover: {skill_dir}/asset/AI日报.png`

**正文格式**：

```markdown
## 📊 今日概览
100-200字总结今日热点事件

---

## 🚀 头条热点

### [标题 1]
**热度指数**：⭐⭐⭐⭐⭐
**信息来源**：[来源]
**发布时间**：[时间]

**事件概述**：
[100-200 字概述事件核心内容]

**关键信息**：
- 关键点 1
- 关键点 2
- 关键点 3

**行业影响**：
[100-200 字分析行业影响]

**深度解读**：
[100-200 字深度分析，包括技术背景、市场反应、未来趋势]

---

##  趋势观察

[100-200 字的今日趋势总结和未来预测]

---

**数据来源**：各公司官网，科技媒体
**整理时间**：YYYY-MM-DD
```

#### 3.3 发布到微信草稿箱

使用 `wenyan-cli` 工具将生成的 Markdown 文章发布到微信公众号草稿箱：

```bash
# 发布科技热点日报
wenyan publish -f {skill_dir}/daily_reports/tech_daily_YYYY-MM-DD.md

# 发布AI热点日报
wenyan publish -f {skill_dir}/daily_reports/ai_daily_YYYY-MM-DD.md
```

## 错误处理

### 常见错误及解决方案

| 错误类型    | 可能原因                      | 解决方案                             |
| ------- | ------------------------- | -------------------------------- |
| 信息源无法访问 | 网络问题或网站维护                 | 使用 `Tavily Search` 补充检索，或切换备用信息源 |
| 微信发布失败  | AppID/AppSecret,IP白名单配置错误 | 检查配置是否正确                         |
| 热点数据不足  | 搜索范围过窄                    | 放宽时间范围至 48 小时，扩大搜索关键词            |
| 工具调用失败  | MCP 服务问题                  | 使用 WebSearch/WebFetch 作为备选方案     |
| 内容抓取超时  | 网页加载慢                     | 减少单次抓取数量，分批处理                    |

### 降级策略

1. **热点检索降级**：trends-hub → Tavily Search → WebSearch
2. **内容采集降级**：Tavily Extract → WebFetch → 简略描述
3. **发布降级**：wenyan publish 失败 → 手动保存 markdown 文件

## 性能优化

### 并行处理

- 同时检索多个 trends-hub 信息源
- 并行抓取多个网页内容
- 批量处理多个搜索查询

### 缓存策略

- 记录已处理的热点事件
- 避免重复抓取相同内容
- 建立热点事件数据库（可选）

## 注意事项

### 信息准确性

- 所有信息必须来自可靠来源
- 重要数据需要多方验证（至少2个独立来源）
- 标注信息来源和发布时间

### 时效性要求

- 优先选择 24 小时内的热点
- 重大事件可放宽至 48 小时
- 标注事件发生时间

### 内容质量

- 避免标题党
- 保持客观中立
- 提供深度分析而非简单罗列

### 微信发布规范

- 标题符合微信规范（不超过64字，不用特殊字符）
- 内容符合微信平台规则
- 图片使用需注意版权
