# Hot Paper WeChat Publish

本技能用于生成高质量的科技行业热点日报和 AI 行业热点日报，并自动发布到微信公众号草稿箱。

## 功能概述

- **热点检索**：全面检索科技领域和 AI 领域的当前热点事件
- **信息采集**：深入搜集每个热点话题的详细信息
- **系统整理**：系统化整理与专业化分析信息
- **日报生成**：生成结构清晰、内容详实的热点日报
- **自动发布**：自动发布到微信公众号草稿箱

## 执行流程

### 第一阶段：热点检索与采集

#### 1.1 检索热点事件

使用 `trends-hub` 工具全面检索当前热点事件：

**筛选标准**：

- 科技领域相关热点
- AI 领域相关热点
- 热度值较高的话题
- 影响力较大的事件

#### 1.2 监控官方渠道

读取并监控以下官方渠道的最新消息：

**信息源文件**：`{skill_dir}/assets/source_links.md`（或在工作目录创建 `company_links.md` 软链接）

**监控范围**：

**国际科技巨头**：Apple、Microsoft、Google、Amazon、Meta、NVIDIA、Samsung、IBM、Intel、TSMC

**国内科技巨头**：百度、华为、腾讯、阿里巴巴、京东、字节跳动

**国际 AI 公司**：OpenAI、Google DeepMind、Anthropic、Mistral AI、Cohere

**国内 AI 公司**：百度文心、智谱 AI、月之暗面 AI、阶跃星辰 AI、零一 AI

**科技媒体**：36Kr、TechNode、GeekPark、雷锋网、新浪科技、TechCrunch、The Verge、Wired

**监控方法**：

1. 从信息源文件（`{skill_dir}/assets/source_links.md`）读取官方链接，或将其复制/软链接到工作目录的 `company_links.md`
2. 使用 运用 `WebSearch` 或`WebFetch` 工具获取各官网的最新动态
3. 筛选出 24 小时内发布的重要消息

#### 1.3 深度信息采集

针对筛选出的每个热点话题，运用 `WebSearch` 和 `WebFetch` 工具深入搜集信息：

**工具使用策略**：

| 工具          | 用途             | 示例            |
| ----------- | -------------- | ------------- |
| `WebSearch` | 广泛搜索相关新闻、报道、评论 | 搜索全网对事件的报道    |
| `WebFetch`  | 抓取具体网页的详细内容    | 获取官网公告、深度文章全文 |

**搜索维度**：

- **事件背景**：事件起因、发展历程
- **核心技术**：技术原理、创新点
- **行业影响**：对行业的影响、市场反应
- **专家观点**：行业专家评价、分析师观点
- **最新动态**：最新进展、后续计划

**操作流程**：

1. **使用** **`WebSearch`** **广泛搜索**
   ```
   搜索关键词示例：
   - "[热点事件] 详细情况"
   - "[公司名称] 最新发布"
   - "[技术名称] 行业影响"
   - "[产品名] 专家评价"
   ```
2. **使用** **`WebFetch`** **抓取关键信息源**
   ```
   抓取目标：
   - 公司官网博客/新闻稿
   - 权威科技媒体深度报道
   - 行业分析报告页面
   ```
3. **信息交叉验证**
   - 对比多个来源的信息
   - 优先采用官方渠道信息
   - 标注信息来源和发布时间

### 第二阶段：信息处理与分析

#### 2.1 信息去重与验证

- 去除重复信息
- 交叉验证信息来源
- 确保信息准确性

#### 2.2 分类整理

**按领域分类**：

- 科技综合
- AI 技术
- 产品发布
- 投融资
- 行业政策

**按重要性分级**：

- S 级（重大）：行业颠覆性事件、巨头核心发布
- A 级（重要）：重要产品更新、重大合作
- B 级（一般）：常规动态、小额融资

#### 2.3 专业分析

- 事件背后的技术趋势
- 对行业格局的影响
- 未来发展方向预测

### 第三阶段：日报编撰与发布

#### 3.1 内容编撰

选择热度最高、影响力最大、最有价值的 10 条事件，分别编撰成：

**科技热点日报** 和 **AI 热点日报**

#### 3.2 日报格式模板

**Front Matter 元信息**（文章开头必须包含）：

```markdown
---
title: 科技热点日报|YYYY-MM-DD
cover: asset/科技日报.png
tags:
  - 科技日报
  - AI
  - 行业动态
summary: 今日科技/AI 行业热点事件汇总，包含 X 条重大发布、X 起投融资事件
---
```

> **提示**：AI 热点日报请使用 `cover: asset/AI日报.png`

**正文格式**：

```markdown

- 热点事件总数：X 条
- 重大发布：X 项
- 投融资事件：X 起
- 行业政策：X 项

---

## 🚀 头条热点

### [标题 1]
**热度指数**：⭐⭐⭐⭐⭐
**信息来源**：[来源]
**发布时间**：[时间]

**事件概述**：
[100-200 字概述事件核心内容]

**关键信息**：
- 关键点 1
- 关键点 2
- 关键点 3

**行业影响**：
[100-200 字分析行业影响]

[100-200 字深度分析，包括技术背景、市场反应、未来趋势]

**信息来源**：[来源]

- **公司**：[公司名称]

- **业务方向**：[业务描述]

- [关注点 3]

---

**数据来源**：各公司官网，科技媒体
**整理时间**：YYYY-MM-DD
```

#### 3.3 发布到微信草稿箱

使用 `wenyan-cli` 工具将生成的 Markdown 文章发布到微信公众号草稿箱：

```bash
# 发布科技热点日报
wenyan publish -f tech_daily_YYYY-MM-DD.md

# 发布 AI 热点日报
wenyan publish -f ai_daily_YYYY-MM-DD.md
```

## 使用方式

### 手动触发

```bash
# 生成并发布今日的科技/AI 热点日报
执行 hot-paper-wechat-publish 技能
```

### 定时任务（建议配置）

在 crontab 中配置每日定时执行：

```bash
# 每天早上 8 点执行
0 8 * * * cd ~/your-project-path && [执行命令]
```

## 输出文件

生成的日报文件将保存在工作目录下的 `daily_reports/` 文件夹：

```
daily_reports/
├── tech_daily_2024-01-15.md
├── ai_daily_2024-01-15.md
└── ...
```

> **提示**：首次使用前请创建 `daily_reports/` 目录

## 注意事项

### 信息准确性

- 所有信息必须来自可靠来源
- 重要数据需要多方验证
- 标注信息来源

### 时效性要求

- 优先选择 24 小时内的热点
- 重大事件可放宽至 48 小时
- 标注事件发生时间

### 内容质量

- 避免标题党
- 保持客观中立
- 提供深度分析而非简单罗列

### 微信发布规范

- 标题符合微信规范（不超过 64 字）
- 内容符合微信平台规则
- 图片使用需注意版权

## 所需工具

- `trends-hub`：热点趋势检索工具
- `Tavily`：搜索引擎工具
- `WebSearch` 和 `WebFetch` ：网页搜索工具
- `wenyan-cli`：微信公众号发布工具
- 标准工具：curl、grep、awk 等
