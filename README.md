# Hot Paper WeChat Publish 🔥

科技/AI 热点日报自动生成与发布工具

## 📖 项目简介

本项目是一个自动化的科技行业热点日报生成系统，能够：
- 🔍 全面检索科技和 AI 领域的热点事件
- 📰 深度采集和整理热点信息
- ✍️ 生成结构清晰、内容详实的热点日报
- 🚀 自动发布到微信公众号草稿箱

## 🎯 核心功能

### 1. 热点检索与采集
- 多维度筛选热点事件（科技/AI 领域）
- 监控官方渠道（科技巨头、AI 公司、科技媒体）
- 深度信息采集（事件背景、核心技术、行业影响）

### 2. 信息处理与分析
- 信息去重与交叉验证
- 分类整理（按领域、重要性分级）
- 专业分析（技术趋势、行业影响、发展预测）

### 3. 日报生成与发布
- 自动生成科技热点日报和 AI 热点日报
- 标准化格式（含 Front Matter 元信息）
- 自动发布到微信公众号草稿箱

### 4. 质量审核（发布前必执行）
- **幻觉内容检测**：识别并修正虚构人物、编造数据、不存在产品等
- **事实准确性核验**：验证时间节点、数据指标、关键人物、事件描述
- **逻辑一致性检查**：确保时间线、因果关系、数据前后一致
- **审核日志记录**：完整记录修改和核验过程

## 🛠️ 使用方式

本工具作为 Trae IDE 的技能（Skill）使用，当需要：
- 生成科技/AI 热点日报
- 自动发布内容到微信草稿箱

系统会自动调用此技能执行任务。

## 📊 日报格式

### Front Matter 元信息
```markdown
---
title: 科技热点日报|2024-01-15
cover: asset/科技日报.png
tags:
  - 科技日报
  - AI
  - 行业动态
summary: 今日科技/AI 行业热点事件汇总
---
```

### 正文结构
- 📊 今日概览（事件统计）
- 🚀 头条热点（S 级重大事件）
- 📈 重要动态（A 级重要事件）
- 📝 其他动态（B 级一般事件）

每条热点包含：
- 热度指数
- 信息来源
- 事件概述
- 关键信息
- 行业影响
- 深度解读

## 📝 注意事项

1. 信息来源优先采用官方渠道
2. 所有信息需交叉验证（至少 2 个独立来源）
3. 标注信息来源和发布时间
4. 遵循微信公众号发布规范
5. **质量审核为发布前的强制性流程，不可跳过**
6. 无法验证的信息必须删除，存疑信息标注"待确认"
7. 审核日志必须完整记录所有修改和核验过程
